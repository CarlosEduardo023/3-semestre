Exemplo de lista de descrição

TCP
O Protocolo de Controle de Transmissão (TCP) é um método (protocolo) usado junto com o Protocolo de Internet (IP) para enviar na forma de unidades de mensagem, chamada de pacotes, entre computadores pela internet.
IP
O Protocolo de Internet (IP) é o método ou protocolo pelo qual os dados são enviados de um computador para outro na internet. Cada computador na Internet é identificado de forma única por um endereço IP.
FPT
O Protocolo de Tranferência de Arquivos (FTP) é um protocolo usado para trocar arquivos entre computadores na internet.
HTTP
O Protocolo de Tranferência de Hipertexto (HTTP) é o protocolo usado para trocar texto, imagens gráficas, com, vídeo e outros arquivos multimídia na Web.
