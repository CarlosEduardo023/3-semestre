A linguagem de programação é uma linguagem esecrita e formal que especifica um conjunto de instruções e regras usadas para gerar programs e softwares.

Reforçando esse conceito, Silas dos Santos Carneiro define:

"Entende-se por linguagem de programação ao conjunto de regras sintáticas definidas pelo usuário e que devem ser executadas pelo computadoe para a realização de tarefas, por exemplo, a solução numérica de problemas matematicos ou aquisição de medidas de experimentos em laboratório."
